## 递归
### 逻辑
自身调用自身
![[递归算法描述.png]]
### 例子
1. 计算 5 的阶乘，5x(4x(3x(2x(1))))；
2. 编写递归函数：
	- func(5)
3. 编写递归调用计算式：（每次函数调用都会保护现场和还原现场，所以不用手动自增或自减操作，栈很适合做这种工作）
	-  5x(...) 即 5 x func(5-1)
4. 确定终止条件：
	- n=1 时到达阶乘边界，1 的阶乘值为 1 ，返回 1；

```cpp
int factorial(int n)
{
	if (n <= 1)
	{
		return 1;
	}
	else
	{
		return n * factorial(n - 1);
	}
}
```
